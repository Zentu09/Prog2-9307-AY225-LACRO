## Prerequisites if node is not installed

1. Install [Node.js](https://nodejs.org/) (current LTS is fine).  
   – The installer includes `npm`; 

2. Open a terminal in the project folder  
   (PowerShell, Cmd, Git Bash, WSL, … whatever you normally use).

3. Run:
  
   npm install      # fetch dependencies listed in package.json
   npm start        # starts the server on http://localhost:4000
